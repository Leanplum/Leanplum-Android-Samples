Instrumenting variables with LEANPLUM

Read related article: https://support.leanplum.com/hc/en-us/articles/209662783-Setup-Instrumenting-variables-iOS-Android-

In this sample we are going to see how to instrument variables that can take on new values from the server. This allows to roll out changes without having to push an update through the store. A/B tests can also be created to change variables for only a percentage of your users.
